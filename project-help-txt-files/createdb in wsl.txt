how to createdb on wsl...


start postgresql by typing in:
sudo /etc/init.d/postgresql start


then use:
sudo -u postgres createdb (dbname)


add tables:
use umbuntu - run:

python3 -i model.py

then type in:
db.create_all() psql storybook

can check database by going through umbuntu or VS code- type in psql storybook 
to get into database, then type \dt to pull up list of tables created...then \d tablename
to get table attributes, then SELECT * from  (tablename) to see everything on table.

drop database:
sudo -u postgres dropdb (dbname)
